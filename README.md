# -BOD-prediction
We will predict the BOD values of 중랑천 river based on the following input data: Temperature, precipitation, air preassure and solar radiation.
-----------------------------------------------------------
Input data:
    - Temperature (ºC)
    - Precipitation (mm)
    - Air preassure (hPA)
    - Solar radiation (Mj/m2)

Output data:
    Biochemical oxygen demand, BOD (mg/L)
